Hikvision DS-K1T341 / DS-K1T343 qurilmalarini BioFace ga toliq ISUP 5.0 asosida ulash tartibi

1. Kamera veb interfeysiga kirish

  • Kamera IP manzilini brauzerda oching (masalan: http://10.10.0.16).
  • Login odatda admin, parol esa qurilma aktivatsiyasida qoyilgan parol.

2. Platform Access - ISUP 5.0 sozlamasi

  1. Network -> Advanced Settings -> Platform Access -> ISUP bolimiga oting.
  2. Enable ni yoqing va Protocol Version = ISUP5.0 qiling.
  3. Server IP Address sifatida BioFace server IP sini kiriting: {{ isup_guide.server_host }}
  4. Port (Register/CMS): {{ isup_guide.register_port }}
  5. Device ID ni BioFace dagi kamera isup_device_id qiymati bilan bir xil kiriting (masalan: CAM1111).
  6. Encryption Key: {{ isup_guide.isup_key }}
  7. Save qiling va Registration Status ni Refresh bilan tekshiring.

3. ISUP Listening (Alarm/Picture) sozlamasi

  1. ISUP Alarm Center IP/Domain Name: {{ isup_guide.server_host }}
  2. ISUP Alarm Center URL: /
  3. ISUP Alarm Center Port: {{ isup_guide.alarm_port }}
  4. Agar qurilmada alohida Picture server maydoni bolsa, uni ham shu server IP va {{ isup_guide.picture_port }} portga moslang.

4. BioFace ichida tekshirish

  • ISUP Server sahifasida quyilarni tekshiring: portlar {{ isup_guide.register_port }}, {{ isup_guide.alarm_port }}, {{ isup_guide.picture_port }} listening holatda bolsin va qurilma Connected royxatida chiqsin.
  • Quick API tekshiruv: {{ isup_guide.health_url }}, {{ isup_guide.process_url }}, {{ isup_guide.devices_url }}.

Muhim eslatmalar

  • ISUP rejimida register, alarm va picture oqimi toliq ISUP kanallari orqali ishlaydi.
  • Kamera serverga chiqishi uchun firewall da TCP {{ isup_guide.register_port }}, {{ isup_guide.alarm_port }}, {{ isup_guide.picture_port }} ochiq bolishi kerak.
  • Server bind manzili {{ isup_guide.bind_host }} bo'lishi normal. Kamera ichiga esa har doim real server IP/DNS yoziladi: {{ isup_guide.server_host }}.
  • Agar Registration Status offline qolsa, avval network mask, gateway va Encryption Key ni qayta tekshiring.
