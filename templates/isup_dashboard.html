Kamerada ISUP ni to'g'ri yoqish uchun qisqa tartib

  1. 1. `Configuration -> Network -> Advanced Settings -> Platform Access` bo'limiga o'ting.
  2. 2. `Enable` ni yoqing va `Protocol` ni `ISUP` qiling.
  3. 3. `Server Address` ga BioFace server IP manzilini yozing.
  4. 4. `Register/CMS` port sifatida odatda `7660` ishlatiladi.
  5. 5. `Alarm` uchun `7661`, `Picture` uchun `7662` tavsiya qilinadi.
  6. 6. `ISUP Key` qiymatini `bioface2024` bilan mos saqlang.
Eslatma: BioFace bu sahifada real Hikvision SDK (`hikvision_sdk`) listenerlari bilan ishlaydi. Register, Alarm va Picture portlari bevosita SDK orqali ochiladi.
