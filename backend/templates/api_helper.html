Command payload

ISUP buyruq yuborish uchun namunaviy JSON

{
  "command": "ping",
  "params": {
    "trace_id": "demo-001",
    "requested_by": "admin"
  }
}

`command` - kamera uchun amal nomi.

`params` - buyruqqa tegishli qo'shimcha maydonlar.

Transport - buyruqlar faqat `isup_redis` orqali yuboriladi; offline qurilmaga yuborish bloklanadi.
